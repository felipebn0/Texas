Comandos GIT:

Solo la primera vez:

1- git config --global user.name nombreuser

2- git config --global user.email email@xxx.com


Para crear carpeta: 

1-mkdir nombreCarpeta

Para cambiar directorio:

1-cd nombreCarpeta 


Para subir archivos:

1- git init

2- touch nombreArchivo (si es una carpeta: touch nombredecarpeta/)

3- git add nombreArchivo (si es una carpeta: git add nombrecarpeta/)

4- git commit -m "nombre referencia" 

5- git remote add origin https://github.com/felipebn0/Texas.git

6- git push -u origin master

-> luego nombre usuario y password 


Para copiar un proyecto al pc:

git clone [url]
